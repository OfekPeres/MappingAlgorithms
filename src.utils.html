KDTree

This class implements the KDTree data-structure which is an efficient organization of points in a k-dimensional space, where each level of the tree is split into a hyperplane that divides the points into two partitions.

Allows functionality to insert points into the tree, return the points in order, and returns the nearest neighbor of a point given another point.
